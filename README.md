# Sistema de Auditoría de Inventario - Restaurante Bendito Sabor

## Descripción
Solución desarrollada en Python para el curso Fundamentos de 
Programación (código 213022) de la Universidad Nacional Abierta 
y a Distancia UNAD - Fase 5 Evaluación Final POA.

El programa audita el inventario de ingredientes del restaurante 
Bendito Sabor, identifica los productos que están por debajo del 
stock mínimo requerido y calcula la cantidad exacta a pedir.

## Ingredientes que maneja
- Pan de hamburguesa
- Carne 125g y Carne 150g
- Tomate y Queso
- Papa criolla
- Plátano verde
- Carne desmechada
- Pollo desmechado
- Salchicha americana

## Tecnología
- Lenguaje: Python 3
- Paradigma: Programación estructurada
- Estructuras: Matrices, Funciones, Ciclos for, Condicionales if

## Cómo ejecutar
1. Tener Python 3 instalado
2. Ejecutar: python bendito_sabor_inventario.py
3. Ingresar el stock actual de cada ingrediente cuando el programa lo solicite
